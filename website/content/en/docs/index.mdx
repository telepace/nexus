# Documentation

Welcome to Nexus Documentation!

## Overview

Nexus is a modern full-stack development platform that provides:

- Backend API development
- Frontend web applications
- Admin dashboard
- Browser extensions

## Quick Start

1. Clone the project
2. Install dependencies
3. Start the development server

```bash
git clone https://github.com/your-repo/nexus.git
cd nexus
pnpm install
pnpm dev
```
