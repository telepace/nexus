# Quick Forge AI 文档

欢迎来到 Quick Forge AI 的官方文档 - 这是一个结合 FastAPI（Python）和 TypeScript 的生产就绪全栈模板，用于快速 AI 原型开发。本文档将帮助您开始使用该项目，了解其架构，并指导您完成开发和部署过程。

## 什么是 Quick Forge AI？

Quick Forge AI 专为需要快速构建和部署具有现代开发实践的专业应用程序的自由职业者和 AI 企业家设计。它提供了一个坚实的基础，结合了：

- 使用 FastAPI 的高性能 Python 后端
- 使用现代 React 和 TypeScript 的类型安全前端
- 为常见 AI 服务预配置的集成
- 具有测试、CI/CD 和部署选项的生产质量架构

## 主要特点

- **FastAPI 后端**：具有自动 OpenAPI 文档的高性能 Python API
- **TypeScript 前端**：使用现代 React 设置的类型安全前端
- **AI 就绪**：为常见 AI 服务和工具预配置集成
- **开发者体验**：针对 Cursor 和其他 AI 驱动的开发工具进行了优化
- **生产质量**：完整的测试框架、CI/CD 流程和部署选项
- **Docker 集成**：容器化开发和部署工作流
- **认证系统**：具有安全密码哈希的 JWT 认证
- **电子邮件集成**：密码恢复和通知系统
- **深色模式支持**：具有浅色/深色主题切换的现代 UI
- **数据库迁移**：使用 Alembic 自动架构管理
