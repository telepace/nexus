# Quick Forge AI 入门指南

本指南将帮助您设置并开始使用 Quick Forge AI。按照以下步骤创建您的开发环境并了解基本项目结构。

## 前提条件

在开始之前，请确保您的系统上已安装以下内容：

- **Python 3.10+**
- **Node.js 18+**
- **Docker 和 Docker Compose**（可选但推荐）
- **Git**

## 安装

### 步骤 1：克隆仓库

首先，将 Quick Forge AI 仓库克隆到您的本地机器：

```bash
git clone https://github.com/telepace/quick-forge-ai.git
cd quick-forge-ai
```

### 步骤 2：设置环境

您可以使用自动脚本设置开发环境：

```bash
./setup.sh
```

或者，您可以手动设置：

#### 后端设置

```bash
cd backend
python -m venv venv
source venv/bin/activate  # 在 Windows 上: venv\Scripts\activate
pip install poetry
poetry install
```

#### 前端设置

```bash
cd ../frontend
pnpm install
```

### 步骤 3：配置

根据提供的 `.env.example` 在根目录中创建一个 `.env` 文件。需要更新的重要值：

```
# 使用以下命令生成安全密钥：
# python -c "import secrets; print(secrets.token_urlsafe(32))"

# 必需的安全设置
SECRET_KEY=您生成的密钥
FIRST_SUPERUSER=admin@example.com
FIRST_SUPERUSER_PASSWORD=您的安全密码
POSTGRES_PASSWORD=您的数据库密码

# 可选的电子邮件设置
SMTP_HOST=
SMTP_USER=
SMTP_PASSWORD=
EMAILS_FROM_EMAIL=info@example.com
```

### 步骤 4：启动开发服务器

使用 Docker Compose 启动所有服务：

```bash
docker-compose up
```

这将启动：
- PostgreSQL 数据库
- 后端 FastAPI 服务
- 前端开发服务器
- Traefik 用于路由（如果已配置）

## 验证设置

服务运行后，您可以通过访问以下内容验证设置：

- 后端 API：[http://localhost:8000](http://localhost:8000)
- API 文档：[http://localhost:8000/docs](http://localhost:8000/docs)
- 前端应用程序：[http://localhost:3000](http://localhost:3000)
- Adminer：[http://localhost:8080](http://localhost:8080)
