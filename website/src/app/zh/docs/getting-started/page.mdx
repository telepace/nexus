# nexus 入门指南

本指南将帮助你设置和运行项目，无论是用于开发还是部署。

## 快速开始使用 Docker Compose

最简单的开始方式是使用 Docker Compose，它将为你设置所有必要的服务。

### 要求

- 已安装 [Docker](https://docs.docker.com/get-docker/)
- 已安装 [Docker Compose](https://docs.docker.com/compose/install/)
- Git

### 安装步骤

1. 克隆仓库：

```bash
git clone https://github.com/telepace/nexus.git
cd nexus
```

2. 设置环境变量（用于开发）：

```bash
cp .env.example .env
```

3. 使用 Docker Compose 启动服务：

```bash
docker compose watch
```

4. 系统启动后，在浏览器中访问：

- 前端 (React)：[http://localhost:3000](http://localhost:3000) 
- 后端 API (FastAPI)：[http://localhost:8000](http://localhost:8000)
- API 文档 (Swagger UI)：[http://localhost:8000/docs](http://localhost:8000/docs)
- API 替代文档 (ReDoc)：[http://localhost:8000/redoc](http://localhost:8000/redoc)
- pgAdmin：[http://localhost:5050](http://localhost:5050)

## 后端开发设置

如果你想单独运行后端，可以按照以下步骤进行：

### 要求

- Python 3.10 或更高版本
- [uv](https://github.com/astral-sh/uv) (Python 包管理器，类似于 pip，但速度更快)

### 安装步骤

1. 安装 uv：

```bash
pip install uv
```

2. 切换到后端目录并创建虚拟环境：

```bash
cd backend
uv venv .venv
source .venv/bin/activate  # 在 Windows 上使用 .venv\Scripts\activate
```

3. 安装依赖项：

```bash
uv pip install -e .
```

4. 运行开发服务器：

```bash
uvicorn app.main:app --reload
```

## 前端开发设置

如果你想单独运行前端，可以按照以下步骤进行：

### 要求

- Node.js 18 或更高版本
- [pnpm](https://pnpm.io/installation) (推荐) 或 npm

### 安装步骤

1. 切换到前端目录并安装依赖项：

```bash
cd frontend
pnpm install
```

2. 运行开发服务器：

```bash
pnpm dev
```

3. 前端将在 [http://localhost:3000](http://localhost:3000) 上可用。
