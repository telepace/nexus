# Quick Forge AI Documentation

Welcome to the Quick Forge AI documentation. Here you'll find all the information about our framework, including getting started guides, best practices, and in-depth tutorials.

## What is Quick Forge AI?

Quick Forge AI is a modern full-stack application framework designed for rapidly building applications with integrated AI capabilities. It brings together the following technology stack:

- **Next.js** frontend framework
- **FastAPI** backend framework
- **PostgreSQL** database
- **Docker** containerization
- **AI Integration** LLM and vector database interfaces
