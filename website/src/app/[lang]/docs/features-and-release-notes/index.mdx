# 功能和发布说明

本页面提供 Quick Forge AI 的主要功能列表和版本更新记录。

## 主要功能

- **现代化全栈框架**：基于 Next.js 和 FastAPI
- **AI 集成**：内置 LLM 和向量数据库支持
- **容器化部署**：完整的 Docker 支持
- **认证系统**：JWT 和 OAuth 支持
- **国际化**：多语言支持
- **主题系统**：明暗主题切换

## 版本历史

### v1.0.0 (2023-10-15)

- 初始版本发布
- 基本的前后端框架结构
- 用户认证系统
- Docker 部署支持
