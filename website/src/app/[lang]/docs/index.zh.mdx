# Quick Forge AI 文档

欢迎来到 Quick Forge AI 文档。这里您可以找到有关我们框架的所有信息，包括入门指南、最佳实践和深入教程。

## 什么是 Quick Forge AI？

Quick Forge AI 是一个现代化的全栈应用框架，专为快速构建集成AI功能的应用而设计。它将以下技术栈整合在一起：

- **Next.js** 前端框架
- **FastAPI** 后端框架
- **PostgreSQL** 数据库
- **Docker** 容器化
- **AI集成** LLM 和向量数据库接口

## 核心特性

- 🚀 **快速启动** - 预配置的开发环境和工具链
- 🔄 **现代化架构** - 使用最新技术栈和最佳实践
- 🧠 **AI功能** - 开箱即用的AI集成
- 🔧 **灵活定制** - 模块化设计，便于扩展
- 📱 **响应式界面** - 漂亮的UI组件和模板
