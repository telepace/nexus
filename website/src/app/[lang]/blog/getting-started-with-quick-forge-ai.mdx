# Quick Forge AI 入门指南

欢迎使用 Quick Forge AI！本指南将帮助您快速了解如何使用这个现代化的全栈框架来构建智能应用。

## 什么是 Quick Forge AI？

Quick Forge AI 是一个集成了最新技术栈的全栈应用框架，专注于快速开发带有AI功能的现代Web应用。它结合了以下技术：

- **Next.js** 前端框架
- **FastAPI** 后端API
- **PostgreSQL** 数据库
- **Docker** 容器化
- **内置AI集成** 包括LLM和向量数据库支持

## 快速开始

按照以下步骤开始使用 Quick Forge AI：

1. 克隆仓库
2. 安装依赖
3. 配置环境变量
4. 启动开发服务器

```bash
# 克隆仓库
git clone https://github.com/username/quick-forge-ai.git
cd quick-forge-ai

# 安装依赖
pnpm install

# 配置环境变量
cp .env.example .env

# 启动开发服务器
pnpm dev
```
